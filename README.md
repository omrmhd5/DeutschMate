# 🎓🇩🇪 DeutschMate

A comprehensive German language learning application built with **Flutter**.
**DeutschMate** helps English users learn German vocabulary through an interactive and engaging interface, featuring multiple categories, audio pronunciations, and visual aids.

---

## ✨ Features

* Multiple Learning Categories – Organized vocabulary sections
* Audio Integration – Native pronunciation for each word
* Visual Learning – Image associations for better retention
* Interactive UI – User-friendly and responsive design

## 🚀 Usage

1. Launch the application
2. Select a category *(Animals, Colors, Family, etc.)*
3. Tap on words to hear pronunciation
4. Practice with interactive excercising

---

## 🛠️ Technologies Used

* Frontend: Flutter, Dart
* Audio: AudioPlayers package
* Styling: Google Fonts, Custom Themes
* Assets: Local image and audio files
